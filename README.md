# BreathIn
Mockup for mobile serious game mobile that supports people during smoking cessation- Bachelor Thesis project 

Full Figma project here: https://www.figma.com/design/Uf1krZ867gHxavWkyZtYCg/MockUp_Bachelor?node-id=0-1&t=6PbIEZe8nCiexaQe-1 

This mockup depicts the possible structure of a serious for smoking cecasstion and the possibilities of how to utilize behaviour change techniques in a serious game. It was designed in the course of my bachelor thesis "Analysis and design study for the implementation of a mobile serious game to support smoking cessation" and was used especially for qualitative research interviews with potential end users as well as a psychologist specialiced in addiction.
